# HARU 캘린더

Android와 Windows에서 일정을 관리하고 동기화하는 무료 캘린더입니다.

- Android 테스트 설치 파일: GitHub Release의 `HARU.apk`
- Play 스토어 제출 파일: `Build Android APK` 작업의 `HARU-play-store-aab`
- Windows 설치 파일: GitHub Release의 `HARU-Setup-*.exe`

출시 준비 문서는 [`docs/PLAY_RELEASE_CHECKLIST.md`](docs/PLAY_RELEASE_CHECKLIST.md)를 참고하세요.
