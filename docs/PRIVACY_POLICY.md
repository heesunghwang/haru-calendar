# HARU 개인정보처리방침

시행일: 2026년 7월 16일

HARU(이하 “앱”)는 사용자의 개인정보를 중요하게 생각합니다. 이 문서는 앱이 어떤 데이터를 처리하고 왜 필요한지 설명합니다.

## 1. 처리하는 데이터

앱은 별도의 회원가입이나 광고 식별자를 사용하지 않습니다. 사용자가 앱에 입력한 다음 정보가 처리될 수 있습니다.

- 일정의 날짜, 이름, 시간, 메모 및 색상
- 날짜 메모
- 사용자가 생성하거나 입력한 12자리 기기 동기화 코드

## 2. 처리 목적과 저장 위치

- 일정과 날짜 메모는 캘린더 기능 제공을 위해 기기 내부 저장소에 저장됩니다.
- 사용자가 동기화 기능을 직접 설정한 경우에만 일정, 날짜 메모 및 동기화 코드가 암호화된 HTTPS 연결을 통해 Supabase 서버에 전송됩니다.
- 서버에 저장된 데이터는 Android와 Windows 기기 사이의 일정 동기화 목적으로만 사용됩니다.

## 3. 제3자 제공 및 광고

앱은 사용자 데이터를 판매하지 않으며 맞춤형 광고를 표시하지 않습니다. 동기화 기능 제공을 위한 클라우드 처리업체로 Supabase를 이용합니다.

## 4. 보관 및 삭제

- 기기 내부 데이터는 사용자가 앱 데이터를 삭제하거나 앱을 제거할 때 삭제됩니다.
- 동기화 연결을 해제하면 해당 기기에서 동기화 코드가 제거되지만, 서버의 기존 데이터는 즉시 삭제되지 않을 수 있습니다.
- 서버 데이터 삭제를 원하는 사용자는 아래 문의처를 통해 삭제를 요청할 수 있습니다. 요청 확인 후 관련 동기화 데이터가 삭제됩니다.

## 5. 보안

앱은 서버 통신에 HTTPS를 사용하고, 기기 간 동기화에는 사용자가 관리하는 12자리 보안 코드를 사용합니다. 사용자는 이 코드를 타인에게 공개하지 않아야 합니다.

## 6. 아동의 개인정보

앱은 아동을 대상으로 개인정보를 의도적으로 수집하지 않습니다. 보호자는 아동이 동기화 기능을 사용할 때 입력하는 일정에 개인정보가 포함되지 않도록 지도해야 합니다.

## 7. 방침 변경

기능 또는 관련 정책이 변경되면 이 문서를 수정하고 시행일을 갱신합니다.
