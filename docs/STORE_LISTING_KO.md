# HARU Google Play 등록 문안

## 앱 이름

HARU

## 짧은 설명

모바일과 Windows에서 함께 사용하는 깔끔하고 편리한 일정 캘린더

## 자세한 설명

HARU는 복잡한 기능 없이 하루의 일정을 간편하게 기록하고 확인할 수 있는 캘린더입니다.

주요 기능

- 월간 캘린더로 일정 확인
- 일정의 날짜, 시간, 메모와 색상 관리
- 공휴일, 생일 등 날짜별 특별 메모
- 일정이 있는 날짜를 한눈에 확인
- Android와 Windows 앱 간 일정 동기화
- 동기화 새로고침과 연결 상태 확인

회원가입 없이 바로 사용할 수 있으며, 동기화 기능은 사용자가 직접 설정한 12자리 보안 코드로 연결됩니다. 광고와 유료 결제 없이 무료로 제공됩니다.

## 분류 제안

- 앱
- 생산성
- 무료
- 광고 포함: 아니요
- 인앱 구매: 아니요

## 데이터 보안 설문 초안

- 데이터를 수집하거나 공유하나요?: 동기화를 사용하는 경우 사용자 생성 콘텐츠를 수집함
- 수집 데이터: 캘린더 일정 및 날짜 메모
- 처리 목적: 앱 기능 및 기기 간 동기화
- 제3자 판매 또는 광고 활용: 하지 않음
- 전송 중 암호화: 예, HTTPS
- 계정 생성: 없음
- 삭제 요청 방법: GitHub Issues의 지원 페이지를 통한 요청

Play Console의 실제 질문 문구가 변경될 수 있으므로 제출 시 앱 동작과 대조해 최종 확인해야 합니다.
